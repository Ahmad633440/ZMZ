ZMZ — Demotivational Quotes

ZMZ is a small Next.js project built for fun.
It serves short demotivational quotes — the kind that feel honest instead of fake-motivating.

Users can submit their own quotes, which are stored in MongoDB and reviewed before being published.

Features

Browse demotivational quotes

Submit quotes for review

Simple, dark UI

MongoDB + Next.js App Router

Deployed on Vercel

Tech Stack

Next.js (App Router)

React

MongoDB Atlas

Tailwind CSS

Vercel

Running Locally 
npm install
npm run dev

Create a .env.local file and add:
MONGO_URI=your_mongodb_connection_string
